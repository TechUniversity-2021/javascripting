# Style Guide
### **Naming Conventions**
<br>

- **filenames**: PascalCase

- **folders**: kebab-case (all lowercase letters) 

- **variables**: camelCase

- **functions**: camelCase

- **classes**: PascalCase

- **boolean**: camelCase
